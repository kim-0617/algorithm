2 단계 : 숫자의 표현

코딩테스트 연습 > 연습문제 > 숫자의 표현

문제 설명

Finn은 요즘 수학공부에 빠져 있습니다. 수학 공부를 하던 Finn은 자연수 n을 연속한 자연수들로 표현 하는 방법이 여러개라는 사실을 알게 되었습니다. 예를들어 15는 다음과 같이 4가지로 표현 할 수 있습니다.

  • 1 + 2 + 3 + 4 + 5 = 15
  • 4 + 5 + 6 = 15
  • 7 + 8 = 15
  • 15 = 15

자연수 n이 매개변수로 주어질 때, 연속된 자연수들로 n을 표현하는 방법의 수를 return하는 solution를 완성해주세요.

제한사항

n은 10,000 이하의 자연수 입니다.

입출력 예
n result
15 4
입출력 예 설명

입출력 예#1
문제의 예시와 같습니다.

코드

function solution(n) {
let answer = 0;
// 2중 for문 돌면서 1부터 시작하는데, i for문에서는 예를들어 1부터 시작했으면 다음에는 1을 제외하고 순차적으로 더해 나가야 겠죠
// j for문에서는 sum에 i 부터 시작하는 수를 순차적으로 더해가면서 sum에 도달했다면 break, 시간을 줄이기 위해 sum을 초과하면 break;
for(let i = 1; i <=n; i++) { 
    let sum=0; 
    for(let j=i; j <=n; j++) { 
        sum +=j; 
        if(sum===n) { 
            answer++; break; 
        } 
        else if(sum>n) break;
    }
}

return answer;
}

리뷰

순서는 다음과 같다
1. 2중 반복문 할게요
2. i = 1부터 시작하죠? i를 1로 고정해놓고 j는 합을 구하는 용도입니다.
3. 1~ 쭉 더해나가다가 n이 되면 break합니다. 그러면 i는 2부터 다시 시작하면서 더하겠죠?
4. 이과정을 반복하면 됩니다. 연속하는 숫자의 합으로 표현되니까요
5. 끝
