2 단계 : 2 x n 타일링

코딩테스트 연습 > 연습문제 > 2 x n 타일링

가로 길이가 2이고 세로의 길이가 1인 직사각형모양의 타일이 있습니다. 이 직사각형 타일을 이용하여 세로의 길이가 2이고 가로의 길이가 n인 바닥을 가득 채우려고 합니다. 타일을 채울 때는 다음과 같이 2가지 방법이 있습니다.

  • 타일을 가로로 배치 하는 경우
  • 타일을 세로로 배치 하는 경우

예를들어서 n이 7인 직사각형은 다음과 같이 채울 수 있습니다.

직사각형의 가로의 길이 n이 매개변수로 주어질 때, 이 직사각형을 채우는 방법의 수를 return 하는 solution 함수를 완성해주세요.

제한사항
  • 가로의 길이 n은 60,000이하의 자연수 입니다.
  • 경우의 수가 많아 질 수 있으므로, 경우의 수를 1,000,000,007으로 나눈 나머지를 return해주세요.
입출력 예
n result
4 5
입출력 예 설명

입출력 예 #1
다음과 같이 5가지 방법이 있다.

코드

function solution(n) {
    const answer = [1, 1];
    
    for(let i = 2; i <= n; i++) {
        const nextValue = answer[i - 2] + answer[i - 1];
        answer.push(nextValue % 1_000_000_007);
    }
    
    return answer[n];
}

리뷰

순서는 다음과 같다
1. n=4 일때는 5가지인데 이건 [1,1,1,1] [1,1,2] [2,1,1] [1,2,1] [2,2] 이렇게 5가지 입니다.
2. 그래서 이런 조합을 생성시키는 코드를 짜서 풀어보려 했는데 효율성에서 충족을 못시키더군요.
3. 그래서 가만보니 4일때는 5, 5일때는 8, 6일때는 13이더군요. 피보나치 수열로 풀면 풀릴 것 같다는 생각이 들었습니다.
4. [1,1] 기본 셋팅하고 i = 2 ~ n까지 증가시키면서 피보나치 수열을 제작합니다.
5. 최종적으로 생성된 피보나치 수열의 n번째 인덱스에 들어있는 값이 정답이 되겠습니다.
6. return 하기전에 나눠주는 것과 answer에 push하면서 나눠주는 행동의 결과가 다른데, 이것은 결과값이 큰 경우에 오버플로우가 나는지 안나는지의 차이점 같습니다.

총평 : 방법은 여러가지가 있을 수 있겠으나, 효율성까지 충족하려면 방법이 많이 제한될 것 같네요.
