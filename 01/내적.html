1 단계 : 내적

코딩테스트 연습 > 월간 코드 챌린지 시즌1 > 내적

문제 설명

길이가 같은 두 1차원 정수 배열 a, b가 매개변수로 주어집니다. a와 b의 내적을 return 하도록 solution 함수를 완성해주세요. 이때, a와 b의 내적은 a[0]*b[0] + a[1]*b[1] + ... + a[n-1]*b[n-1] 입니다. (n은 a, b의 길이)

제한사항

a, b의 길이는 1 이상 1,000 이하입니다. a, b의 모든 수는 -1,000 이상 1,000 이하입니다.

입출력 예
a b result
[1,2,3,4] [-3,-1,0,2] 3
[-1,0,1] [1,0,-1] -2
입출력 예 설명

입출력 예 #1
a와 b의 내적은 1*(-3) + 2*(-1) + 3*0 + 4*2 = 3 입니다.
입출력 예 #2
a와 b의 내적은 (-1)*1 + 0*0 + 1*(-1) = -2 입니다.

코드

function solution(a, b) {
let answer = 0;
for(let i = 0; i < a.length; i++){
    answer += a[i]*b[i]; // 두 배열간의 같은 인덱스에 있는 원소끼리 곱해서 더해준다.
}
return answer;
}

리뷰

순서는 다음과 같다
1. 두 배열의 길이가 같으므로 배열의 원소 갯수만큼 순회하면서 같은 요소의 있는 것들 끼리 곱해서,
2. answer에 누적해주면 된다. 그 후 return.
