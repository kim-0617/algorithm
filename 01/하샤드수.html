1 단계 : 하샤드 수

코딩테스트 연습 > 연습문제 > 하샤드 수

양의 정수 x가 하샤드 수이려면 x의 자릿수의 합으로 x가 나누어져야 합니다. 예를 들어 18의 자릿수 합은 1+8=9이고, 18은 9로 나누어 떨어지므로 18은 하샤드 수입니다. 자연수 x를 입력받아 x가 하샤드 수인지 아닌지 검사하는 함수, solution을 완성해주세요.

제한 조건

x는 1 이상, 10000 이하인 정수입니다.

입출력 예
arr return
10 true
12 true
11 false
13 false
입출력 예 설명

입출력 예 #1
10의 모든 자릿수의 합은 1입니다. 10은 1로 나누어 떨어지므로 10은 하샤드 수입니다.

입출력 예 #2
12의 모든 자릿수의 합은 3입니다. 12는 3으로 나누어 떨어지므로 12는 하샤드 수입니다.

입출력 예 #3
11의 모든 자릿수의 합은 2입니다. 11은 2로 나누어 떨어지지 않으므로 11는 하샤드 수가 아닙니다.

입출력 예 #4
13의 모든 자릿수의 합은 4입니다. 13은 4로 나누어 떨어지지 않으므로 13은 하샤드 수가 아닙니다.

코드

function solution(x) {
    // split해서 자리수 합 구하고, 그걸 x % 자릿수합 한게 나누어떨어지면 true, 아니면 false return
    return x % String(x).split('').reduce((prev, cur) => prev += Number(cur), 0) === 0 ? true : false;
}

리뷰

순서는 다음과 같다
1. String.split('') 하면 ['1', '0'] 이렇게 되겠죠? (10이면) 거기서 reduce 분여서 합구해줄게요
2. 그 과정에서 Number(cur) 해줘서 감쪽같이 바꿔줍니다. 합이 나오면 그 수를 다시 10으로 나눠 봐야 겠죠?
3. 나머지가 0이면 true, 아니면 false
총평 : 이렇게 한줄로 써질 때가 제일 기분 좋습니다.
