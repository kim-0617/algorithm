문제 설명

2016년 1월 1일은 금요일입니다. 2016년 a월 b일은 무슨 요일일까요? 두 수 a ,b를 입력받아 2016년 a월 b일이 무슨 요일인지 리턴하는 함수, solution을 완성하세요. 요일의 이름은 일요일부터 토요일까지 각각 SUN,MON,TUE,WED,THU,FRI,SAT입니다. 예를 들어 a=5, b=24라면 5월 24일은 화요일이므로 문자열 "TUE"를 반환하세요.

제한 조건

2016년은 윤년입니다. 2016년 a월 b일은 실제로 있는 날입니다. (13월 26일이나 2월 45일같은 날짜는 주어지지 않습니다)

입출력 예
a b result
5 24 "TUE"

코드

function solution(a, b) {
    let answer = '';
    let weekday = ["FRI","SAT","SUN","MON","TUE","WED","THU"]; // 요일
    let calender = { // 날짜 수
        1 : 31,
        2 : 29,
        3 : 31,
        4 : 30,
        5 : 31,
        6 : 30,
        7 : 31,
        8 : 31,
        9 : 30,
        10 : 31,
        11 : 30,
        12 : 31,
    }
    for(let i = 1; i < a; i++){
        b += calender[i]; // 총 몇일이 지났는가?
    }
    // 1월 1일은 금요일인데, 1월 1일에서 0일 지나면 금요일, 1일 지나면 토요일, 2일 지나면 일요일... (배열의 인덱스기 때문에 1빼줍니다.)
    return weekday[(b-1)%7]; 
}

리뷰

순서는 다음과 같다
1. 1~12월까지 총 몇일이 있는가 객체로 관리합니다.
2. 1/1일로 부터 몇일이 지났는가 b에 더해줍니다.
3. 금요일부터 시작했고, 배열의 인덱스기 때문에 (b-1) % 7 해주면 b일이 지난 후의 요일이 나올겁니다.
