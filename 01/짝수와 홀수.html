1 단계 : 짝수와 홀수

코딩테스트 연습 > 연습문제 > 짝수와 홀수

정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 반환하는 함수, solution을 완성해주세요.

제한 조건

num은 int 범위의 정수입니다. 0은 짝수입니다.

입출력 예
num return
3 "Odd"
4 "Even"

코드

function solution(num) {
    return num % 2 ? "Odd" : "Even";
}

리뷰

순서는 다음과 같다
1. 2로 나누어 떨어지면 짝수고 나머지가 1이면 홀수잖아요?
