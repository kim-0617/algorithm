1 단계 : 약수의 개수와 덧셈

코딩테스트 연습 > 월간 코드 챌린지 시즌2 > 약수의 개수와 덧셈

문제 설명

두 정수 left와 right가 매개변수로 주어집니다. left부터 right까지의 모든 수들 중에서, 약수의 개수가 짝수인 수는 더하고, 약수의 개수가 홀수인 수는 뺀 수를 return 하도록 solution 함수를 완성해주세요.

제한사항
1 ≤ left ≤ right ≤ 1,000

입출력 예
left right result
13 17 43
24 27 52
입출력 예 설명
입출력 예 #1
다음 표는 13부터 17까지의 수들의 약수를 모두 나타낸 것입니다.
수 약수 약수의 개수
13 1, 13 2
14 1, 2, 7, 14 4
15 1, 3, 5, 15 4
16 1, 2, 4, 8, 16 5
17 1, 17 2
따라서, 13 + 14 + 15 - 16 + 17 = 43을 return 해야 합니다.

입출력 예 #2
다음 표는 24부터 27까지의 수들의 약수를 모두 나타낸 것입니다.
수 약수 약수의 개수
24 1, 2, 3, 4, 6, 8, 12, 24 8
25 1, 5, 25 3
26 1, 2, 13, 26 4
27 1, 3, 9, 27 4
따라서, 24 - 25 + 26 + 27 = 52를 return 해야 합니다.

코드

// 풀이를 보고 느낀점은 Math.sqrt()를 했을 때 정수면 약수의 갯수가 홀수라는 점이다.
// 그렇게 푼다면 2중 반복문이 필요없어서 멋진 풀이였습니다.
// 무난하게 푼 것 같아요

function solution(left, right) {
    let answer = 0; // return 할 변수
    for(let i = left; i <= right; i++){ // left ~ right 까지
        let count = 0; // 약수의 갯수를 판별할 변수
        for(let j = 1; j <= i; j++){ // 1~자기자신까지 순회하면서 나누어 떨어지면 count 증가
            if(!(i%j)){
                count++;
            }
        }
        if(count % 2 === 0){ // 약수의 갯수가 짝수이면 더해주고
            answer += i;
        }
        else{ // 홀수이면 빼준다
            answer -= i;
        }
    }
    return answer;
}

리뷰

순서는 다음과 같다
1. left~right 까지 순회하면서
2. 약수의 갯수(나누어 떨어지는 수)가 짝수이면 더해주고 홀수이면 빼줍니다.

총평 : Math.sqrt()가 정수이면 약수의 갯수가 홀수라는 사실을 이용하면 더 멋진 풀이가 되겠죠?
